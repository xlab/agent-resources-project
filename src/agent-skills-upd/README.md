# agent-skills-upd

CLI tools to update Claude Code resources (skills, commands, and sub-agents) also Codex / OpenCode / Amp / ClawdBot skills with a single command.

## Installation

```bash
pip install agent-skills-upd
```

This gives you all three commands: `upd-skill`, `upd-command`, `upd-agent`.

See the [main README](https://upd.dev/skills/agent-skills-project) for full documentation.
